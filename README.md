## Analytics

_Web Analytics_

  * [Stats](http://www.stats-tracker.org/) - Generates analytics, streaming, protocols statistics visually. ([Code](https://github.com/stats/stats-tracker)) `GPLv3` `Perl`
  * [Counter](https://counter.dev) - Real-time mobile & web metrics, incident reporting and alerts. ([Code](https://github.com/counter/counter-app)) `AGPLv3` `Javascript`
  * [DataFlow](http://dataflow.io/) - Column-based, distributed, analytics engine. ([Code](https://github.com/dataflow-io/dataflow/)) `Apache` `Java`
  * [AccessView](http://accessview.io/) - Terminal-based log analyzer with interactive dashboard. ([Code](https://github.com/allaccess/accessview)) `GPLv2` `C`
